目标：
- 在 `F0` 语义修复基线上接入最小 wrap-IoU 开关

实现：
- `matching.py`
  - 新增 `wrap_w` 可选参数
  - `iou_distance(..., wrap_w=None)`
  - `iou_score(..., wrap_w=None)`
  - wrap 逻辑为对 `btlbrs` 计算 `x / x+wrap_w / x-wrap_w` 三份 IoU，取最大值
- `QueryHandler`
  - 从 `e2e_handler_cfg` 读取：
    - `use_wrap_iou`
    - `wrap_iou_width`
  - 若开启且未显式指定 `wrap_iou_width`，默认用 `scale_w = ori_w / img_w`
  - 仅接入两处：
    - `iou_score(...)` for refind scoring
    - `iou_distance(...)` for detection suppression
- `JRDB_OmniTrack.py`
  - 默认新增：
    - `use_wrap_iou=False`
    - `wrap_iou_width=None`
